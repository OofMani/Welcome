Project 1:

Super Mario Bros!


I created a Deep Q learning algorithm for an AI agent to solve Super Mario Bros. The neural network rewards the agent for progressing forward in the least time. This is a video showcasing the earlier generations of the system. Through this project, I gained a thorough understanding of reinforcement learning and incorporating AI agents in game design.

The full codebase for this project can be found here.

Project 2:

Reckless Driving

I designed a self-driving algorithm implementaion that drives a car super fast and avoids other cars on the highway. The project uses a genetic neural network to learn how to avoid cars. This project was built entirely in JavaScript without using any libraries. This project uses a very intuitive approach to Neural Network Design, and greatly furthered my understanding of the same.

The full codebase for this project can be found here. You can also get a live demo right on your browser using this link.
